# Global Sales & Profit Analysis Dashboard

An interactive dashboard to analyze $119M+ in global sales data. 
Built using Power BI for data modeling and React for the frontend.

## 📊 Key Insights
- **$119M** Total Sales | **$17M** Total Profit | **13.21%** Profit Margin
- Sales peaked in **Oct-Nov**, declined towards **Apr-May**
- Top Segment: **Government** - 44.22% of sales
- Top Product: **Paseo** - 27.87% of revenue
- Top Country: **United States of America**
- Dynamic filters for Country, Month, and Segment

## 🛠️ Tech Stack
- **Power BI** - Data Modeling & Visualization
- **React.js** - Frontend Framework
- **Recharts** - Interactive Charts
- **HTML/CSS** - Styling
